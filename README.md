## 📌 프로젝트 소개

**대한민국 밤 산책**은 경복궁 별빛야행 축제를 소개하고 홍보하기 위해 제작한 웹 프로젝트입니다.  
메인 축제 소개뿐 아니라 유사한 축제 정보도 함께 탐색할 수 있도록 구성했으며,  
`Three.js`와 `Blender`를 활용해 보다 생동감 있는 사용자 경험을 제공하고자 했습니다.

## ✨ 주요 구현 내용

### 1. 자주하는 질문(FAQ)
- **검색 기능**
  - 검색어 입력 후 버튼 클릭뿐 아니라 `Enter` 키로도 검색이 가능하도록 구현
  - 검색 결과에 따라 리스트 개수가 줄어들면, 페이지네이션 개수도 자동으로 변경되도록 처리

- **토글 기능**
  - 질문 클릭 시, 클릭한 항목의 `id`와 데이터 `id`를 비교해 해당 항목만 열리고 닫히도록 구현

<br />

### 2. 마이페이지 - 1:1 문의
- **메인 리스트**
  - 로그인한 사용자와 동일한 아이디를 가진 문의 내역만 출력되도록 구현
  - 리스트 클릭 시 상세 페이지로 이동해 내용을 확인할 수 있도록 구성
  - 페이지네이션 컴포넌트를 적용해 한 페이지당 10개씩 출력되도록 처리
  - 최신 문의가 상단에 보이도록 **역순 정렬**해 사용자 편의성을 높임

![1:1 문의 메인](https://github.com/user-attachments/assets/1b434c0e-c311-4910-b851-980077b3d6eb)

- **상세 페이지**
  - 목록 버튼을 추가해 문의하기 메인 페이지로 쉽게 이동할 수 있도록 구현
  - 수정 / 삭제 버튼을 통해 각 문의 항목을 직접 관리할 수 있도록 구현

- **문의 수정 페이지**
  - 문의 내용을 불러와 수정할 수 있도록 구현
  - 수정 완료 / 취소 버튼 클릭 시 알림이 노출되도록 처리

![1:1 문의 수정 페이지](https://github.com/user-attachments/assets/c7355232-8b1d-477e-a155-6438607356ba)

- **문의 작성 페이지**
  - 메인 리스트의 글쓰기 버튼을 통해 문의 작성 페이지로 이동할 수 있도록 구현
  - 작성 완료 시 알림이 노출되고, 이후 문의하기 메인 리스트 페이지로 이동
  - 작성 완료된 문의 내역이 최상단에 보이도록 구성

![1:1 문의 작성 페이지](https://github.com/user-attachments/assets/a86c2447-2ca9-4eb0-9580-5e9531ee08e1)

<br />

### 3. 기타 구현
- `react-paginate` 라이브러리를 사용해 여러 페이지에서 재사용 가능한 **공용 페이지네이션 컴포넌트**를 제작
- 전체 페이지를 반응형으로 구현해 다양한 기기에서도 일관된 사용자 경험을 제공할 수 있도록 구현

## 💭 트러블슈팅

### 1. 데이터 전달 구조 문제
`router`를 사용해 페이지를 이동시키면서, 필요한 데이터를 `App.jsx`를 통해 전역적으로 전달하려고 하였으나,
여러 데이터의 흐름이 한 파일에 모이면서 데이터가 뒤엉키거나 구조가 복잡해지는 문제가 발생.

해결을 위해 `1:1 문의` 페이지에서는 단순 라우터 이동 방식 대신,  
**`useState`와 `createContext`를 활용해 페이지 전환과 데이터 전달을 처리.**
그 결과 필요한 독립적인 데이터 관리가 가능했고, 다른 팀원의 작업과 충돌하지 않도록 개선.

## 🌱 배운 점

### initial
- CRUD 기능 구현을 통해 데이터의 흐름과 상태 변화의 전 과정을 직접 설계하며 웹 애플리케이션의 기본 구조를 체득
- 검색, 토글, 페이지네이션, 작성/수정/삭제 기능을 연결하면서 게시판 구조를 실제로 경험
- 팀 프로젝트에서 데이터 전달 방식과 역할 분리가 코드의 가독성과 생산성에 미치는 직접적인 영향을 경험

### 2026 update
- 초기 설계가 리팩토링 비용을 얼마나 증감시키는지 체험하며, 확장성을 고려한 설계의 중요성을 체감
- 단순히 '작동하는 코드'를 넘어, 추후 쉽게 읽고 수정할 수 있는 유지보수성 높은 코드를 작성하는 법을 경험

- 초기 설계 시 상태 관리 규칙이 일관되지 않아 유지보수 난이도가 상승.
